# @focusmcp/codebase

Complete codebase intelligence — loads treesitter, symbol, outline, callgraph, depgraph, refs.

## Type

Composite (thin aggregator — no own tools).

## Dependencies

| Brick | Prefix | What it provides |
|-------|--------|-----------------|
| treesitter | ts | Code indexing and parsing |
| symbol | sym | Symbol lookup |
| outline | out | File and repo structure |
| callgraph | cg | Call graph analysis |
| depgraph | dep | Dependency graph analysis |
| refs | refs | Cross-references |
