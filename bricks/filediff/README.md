## Tools

| Tool | Exposed as | Description |
|------|-----------|-------------|
| `diff` | `fd_diff` | Compare two files and return unified diff |
| `patch` | `fd_patch` | Apply a unified diff patch to a file |
| `delta` | `fd_delta` | Show only changed lines between two strings (compact) |
