# @focus-mcp/fileops

File operations for FocusMCP — move, copy, delete, rename.

## Tools

| Tool | Exposed as | Description |
|------|-----------|-------------|
| `move` | `fo_move` | Move file or directory |
| `copy` | `fo_copy` | Copy a file |
| `delete` | `fo_delete` | Delete file or directory |
| `rename` | `fo_rename` | Rename within parent dir |
