# @focusmcp/filesystem

File system operations for FocusMCP — read, write, list, search, and manage files.

## Tools

| Tool | Description |
|------|-------------|
| `fs_read` | Read a file |
| `fs_write` | Write/create a file |
| `fs_list` | List directory contents |
| `fs_search` | Search for patterns (grep) |
| `fs_delete` | Delete a file |

## Installation

```bash
focus add filesystem
```
