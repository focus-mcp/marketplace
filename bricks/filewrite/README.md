# @focus-mcp/filewrite

Write files for FocusMCP — create, overwrite, append.

## Tools

| Tool | Exposed as | Description |
|------|-----------|-------------|
| `write` | `fw_write` | Create or overwrite a file |
| `append` | `fw_append` | Append to end of file |
| `create` | `fw_create` | Create only if not exists |
