# @focusmcp/graphbuild

Knowledge graph builder — construct a graph of code entities (files, functions, types) and their relationships.

## Tools

| Tool | Exposed as | Description |
|------|-----------|-------------|
| `build` | `gb_build` | Build a knowledge graph from a directory |
| `update` | `gb_update` | Incrementally update the graph for changed files |
| `watch` | `gb_watch` | Return graph stats and detect stale nodes |
| `add` | `gb_add` | Manually add a node or edge to the graph |
| `multimodal` | `gb_multimodal` | Return the graph in multiple formats |
