# @focus-mcp/impact

Change impact analysis — given a file, find what files are affected by a change.

## Tools

| Tool | Exposed as | Description |
|------|-----------|-------------|
| `analyze` | `imp_analyze` | Analyze impact of changing a file |
| `affected` | `imp_affected` | List all files affected by changes to given files |
| `propagate` | `imp_propagate` | Trace how a change propagates through the dep graph |
