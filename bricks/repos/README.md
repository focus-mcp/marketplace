# @focusmcp/repos

Multi-repo management — register, track, and get stats for multiple repositories. Configuration stored in `~/.focus/repos.json`.

## Tools

| Tool | Exposed as | Description |
|------|-----------|-------------|
| `list` | `repos_list` | List all registered repositories |
| `register` | `repos_register` | Register a repository path by name |
| `unregister` | `repos_unregister` | Remove a registered repository |
| `stats` | `repos_stats` | File and line count statistics for a repo |
