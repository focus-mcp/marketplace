# @focusmcp/savings

Track and report token savings — compare FocusMCP usage vs naive full-read, trend over time, ROI.

## Tools

| Tool | Exposed as | Description |
|------|-----------|-------------|
| `report` | `sav_report` | Generate a savings report for a session |
| `compare` | `sav_compare` | Compare two sessions' efficiency |
| `trend` | `sav_trend` | Show savings trend over multiple sessions |
| `roi` | `sav_roi` | Calculate return on investment |
