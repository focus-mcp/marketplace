# @focusmcp/task

Task management for multi-agent workflows — create tasks, assign to agents, track status, mark complete.

## Tools

| Tool | Exposed as | Description |
|------|-----------|-------------|
| `create` | `tsk_create` | Create a task with title, description, and priority |
| `assign` | `tsk_assign` | Assign a task to an agent by ID |
| `status` | `tsk_status` | Get task details by ID or list tasks filtered by status |
| `complete` | `tsk_complete` | Mark a task as complete with an optional result |
