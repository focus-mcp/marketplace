# @focusmcp/thinking

Structured reasoning chains — think step by step, branch alternatives, revise conclusions, summarize.

## Tools

| Tool | Exposed as | Description |
|------|-----------|-------------|
| `think` | `thk_think` | Add a reasoning step to the current chain |
| `branch` | `thk_branch` | Explore an alternative line of thinking |
| `revise` | `thk_revise` | Revise a previous step, keeping audit trail |
| `summarize` | `thk_summarize` | Summarize the chain into a structured conclusion |
